# sga1-build-main

## When to use
Fifth step of the SGA1 normalization pipeline, after [[sga1-split-chapters]]. Produces the entry-point `main.tex` and a slim macros package.

## How to run
```
python3 .claude/skills/sga1-build-main/build_main.py 01-normalized_tex/
```

## Output
- `01-normalized_tex/main.tex` — modern minimal preamble (`book` class; `amsmath`/`amssymb`/`amsthm`, `mathrsfs`, `xy`, `makeidx`, `hyperref`, French `babel`), `\input{chapter-NN}` lines auto-generated by globbing chapter files, framed by `\frontmatter`/`\mainmatter`/`\backmatter` and `\maketitle`/`\tableofcontents`/`\printindex`.
- `01-normalized_tex/sga1-macros.sty` — slim package with the SGA1 theorem environments, math operators (`\Spec`, `\SheafHom`, `\Pic`, `\Gal`, …), blackboard/fraktur/sheaf shortcuts, old-font math switches (`\rm`/`\it`/`\bf`/`\cal`), French typographic helpers, the **two-argument** `\thebibliography` (per-exposé bibliographies), and the **corrected-edition** forms of the orig-conditional macros (`\fets`, `\nieme`, `\Gl`→`\mathbf{GL}`, …). Stripped of: hyperref/SMF-class patches, the SMF title page, the `l → ℓ` and active-paren catcode tricks.

## Keep in sync
The macro set here mirrors the expansion tables (`MATH_0ARG`/`MATH_1ARG`/`MATH_NARG`, `TEXT_MACROS`, `THEOREM_CAPTIONS`) in [[sga1-convert-html]]'s `convert.py`. When you add or change a content macro, update both.
