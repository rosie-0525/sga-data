# sga2-build-main

## When to use
Fifth step of the SGA2 normalization pipeline, after `sga2-split-chapters`. Produces the entry-point `main.tex` and a slim macros package.

## How to run
```
python3 .claude/skills/sga2-build-main/build_main.py 01-normalized_tex/
```

## Output
- `01-normalized_tex/main.tex` — modern minimal preamble, `\input{chapter-NN}` lines auto-generated by globbing chapter files.
- `01-normalized_tex/sga2-macros.sty` — slim package with theorem environments, math operators, calligraphic/bold/sheaf letter shortcuts, and French typographic helpers. Stripped of: hyperref patches, \sisi machinery, editorial apparatus (\nde, \pageoriginale, \sfootnote), SMF title/bibliography styling, and active-paren catcodes (stripped for the HTML converter).
